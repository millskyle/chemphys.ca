# Publications

In `publications.js`, a publications entry looks like

```js
            {
                authors: ["First Last", "First Last"],
                title: "Title strings",
                date: "YYYY-MM-DD",
                DOI: "10.1109/5.771073",
                arxiv: "1710.08053",
                abstract: "Optional, unused as of now",
            },
```
